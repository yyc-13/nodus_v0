![](https://imgur.com/g4YZzcI.png)

### Backend

- Environment: Linux + **Node.js**
- Framework: **Express.js**
- User Authorization: **JSON Web Tokens**
- Server-side Renderer: **EJS**

### Front-End

- HTML
- CSS
- JavaScript + **Fetch** + **jQuery**

### Database

- **RDS** + **MySQL**
<!-- - ORM: **Sequelize** -->

  ![](https://imgur.com/u1af0ZE.png)

- Protocol: **HTTP & HTTPs**
- DNS: **GoDaddy**
- Proxy server: **Nginx**

- Test: **Jest**

- **S3**
